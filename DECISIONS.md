Decisions made during the 2026-04-10 session. This file serves as a reference for future development.

## Hosting & Deployment

- **Hosting**: Cloudflare Pages (static site)
- **Domain**: `slowdoctor.dev` ‚Äî registered under personal Cloudflare account (separate from clinic account for `lead.ps`/`leadps.co.kr`)
- **GitHub repo**: `slowdoctor-dev/slowdoctor-web` (public)
- **Git remotes**: `origin` = Gitea (`git.lead.ps`), `github` = GitHub
- **Deploy trigger**: Push/merge to `main` on GitHub triggers automatic Cloudflare Pages build
- **Build settings**: command `npm run build`, output directory `out`
- **Preview deploys**: Every PR gets an automatic preview URL (`*.slowdoctor-web.pages.dev`)

## Git Workflow

- **Branch ‚Üí PR ‚Üí merge** for all changes
- Direct push to `main` is not used (branch protection recommended)
- Initial setup commits may go directly to `main` as a one-time exception

## Content Pipeline

```
lead/ creative-content (MD) ‚Üí slowdoctor-web/src/content/incoming/ ‚Üí npm run convert ‚Üí MDX ‚Üí git push ‚Üí PR ‚Üí merge ‚Üí deploy
```

- Blog posts are written in `lead/` repo by creative-content agent (channel code: SD-WB)
- Confirmed Markdown files are copied to `src/content/incoming/`
- `npm run convert` transforms MD to MDX with proper frontmatter
- If `axes` or `tags` are missing, the convert script calls Claude Haiku API to auto-classify
- `ANTHROPIC_API_KEY` env var required for auto-classification; graceful skip if absent

## Blog Classification

- **No categories** ‚Äî posts are classified by axes + tags only
- **Axes**: `physician` (medicine/clinical) / `engineer` (tech/dev) / `life` (personal/philosophy) ‚Äî three integers, must sum to 10
- **Tags**: Free-form English lowercase strings, 3-5 per post
- Both can be manually set in frontmatter or auto-generated by Claude Haiku during conversion

## Blog Profile (SD-WB)

- **Language**: English
- **Persona**: Inherits slowdoctor (Joonho Lim) ‚Äî honest, non-promotional, first-person
- **Tone**: Personal essay, not academic. Sharp and direct, like the Korean Threads voice but in English
- **Purpose**: Personal blog for topics that don't fit other channels ‚Äî dev logs, clinic operation reflections, personal interests
- **Rules**: No direct LEAD mention, no translation of Korean channel posts, medical advertising law compliance
- **Profile file**: `lead/creative-content/persona-slowdoctor/profiles/SD-WB.md`

### Content pipeline reference

Blog content comes from `lead/creative-content/persona-slowdoctor/outputs/SD-WB/`. The workflow:

1. Post is written and confirmed in `lead/`
2. Copy the `.md` file to `src/content/incoming/` in this repo
3. `npm run convert` ‚Äî transforms to MDX, auto-generates axes/tags if missing
4. Create a branch, commit, push, open PR
5. Verify preview URL
6. Merge ‚Üí auto-deploy to production
