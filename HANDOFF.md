## Important Repo Notes

- Read `AGENTS.md` before making framework-level changes. This repo uses Next.js 16 and the local docs in `node_modules/next/dist/docs/` are the source of truth.
- `next/font/google` is still used in `src/app/layout.tsx`. Builds need network access for Google Fonts unless fonts are vendored locally.
- `public/sitemap.xml` is a generated artifact. Do not hand-edit it.
- Blog frontmatter currently supports:
  - `title`
  - `date`
  - `description`
  - `image?`
